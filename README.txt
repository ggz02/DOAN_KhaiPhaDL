Thạc sĩ Nguyễn Thị Trần Lộc, Source Code được thực hiện bởi Nhóm 2 gồm các sinh viên:
1. Nguyễn Trà Giang - 2121001008
2. Phạm Minh Tâm - 2121005223

Để khởi chạy Source Code, sau khi cài đặt RStudio thành công trên máy tính, chọn mở file: Nhom2_C5.Rproj
Lúc này, project RStudio do nhóm 2 tạo sẽ được khởi chạy tự động (cùng với các file có sẵn trong folder) trên RStudio.

File 3ThuatToan.R: Lưu tất cả các hàm code về tiền xử lý + chạy thuật toán + visualize của 3 thuật toán phân lớp, gom cụm, kết hợp.
Ban đầu file được lưu với tên phanlop.R (được chụp trong Word), sau đó được đổi tên thành 3ThuatToan.R để phù hợp với nội dung của file.
